Gradient Descent¶

Gradient Descent is an algorithm that minimizes some objective $J(\theta)$ where $\theta$ is $\in \mathbb{R}^d$ by iteratively updating the vector $\theta$ by moving in the direction of steepest descent multiplied by some step size or learning rate.

In essence,

$\>\>\>\>$1: Pick an initial guess $\theta_0$
$\>\>\>\>$2: $\>\>$while $J(\theta_{k+1}) - J(\theta_{k}) \leq precision$:
$\>\>\>\>$3: $\>\>\>\>\>\>\>\>$ $\theta_{k+1}$ = $\theta_{k} -\nabla J_\theta(\theta_k)$ * $\alpha_k$.

This is relevant to Machine Learning because it offers a numerical solution to estimates of parameters that do not have a closed form solution.

Estimating Multinomial Logistic Regression Parameters with Gradient Descent¶

One needs to determine the objective $J(\theta)$ before executing this algorithm. In the case of linear regression, the sum of squared residuals could be used. In this case, one can use the idea of MLE to minimize the log liklihood.

For $k \in K$ where $K$ is the set of labeled classes, $p$ is the number of features, and $x_i \in \mathbb{R}^d$

In the case where $K = \{0,1\}$, the model was defined as $p(x_i) = \sigma(\textbf w^T \textbf x_i + b)$ and the likelihood function was $\prod_{i=1}^{n} p(\textbf x_i)^{y_i} (1-p(\textbf x_i))^{1-y_i}$. The output of such a model is simply a probability of a input belonging to class k = 1.

In the case where $|K| > 2$,

Let $p_k(\textbf x)$ be the posterior probability $P(Y = k | X = \bf x)$ and the softmax function which bounds to the output to [0,1]

\begin{equation} p_k(\textbf x_i) = \frac{e^{\beta_{0}^k + \textbf w_k\cdot \textbf x}}{\sum_{j=1}^{K} e^{\beta_{0}^j + \textbf w_j\cdot \textbf x}} \end{equation}

Here, the output of such a model is a vector which estimates how the probability of a input vector $\textbf x_i$ belongs to each class $1...K$. Mathematically that is,

\begin{equation} p_{\textbf w}(\textbf x_i) = \begin{bmatrix} P(Y = 1 | X = \bf x_i) \\ . \\ . \\ P(Y = K | X = \bf x_i) \end{bmatrix} = \begin{bmatrix} p_1(\textbf x_i) \\ . \\ . \\ p_K(\textbf x_i) \end{bmatrix} \end{equation}

One can see how each class $1 \leq k \leq K$ has its own set of parameters $\textbf w_k = (\beta_0,...,\beta_p)$.

We call the K sets of parameters to learn $\textbf w = \begin{bmatrix} \textbf w_1 & . & . & \textbf w_K \end{bmatrix}$, a $d$ by $k$ matrix

Cost Function¶

We want the optimal set of parameters(weights): argmax $J(\textbf w$ $|$ $ \textbf x)$

$$ \prod_{k=1}^{K}P(Y = k | X = \textbf x_i) = \prod_{k=1}^{K} k_i * p_k(\textbf x_i)$$

where $k_i$ is 1 if $\textbf x_i$ is labeled as class k and 0 otherwise

Minimizing the negative log-likelihood is the same as maximizing the log-liklihood and can be taken as an average over an entire sample of size $n$

$$J(\textbf w) = \frac{-1}{n}log\prod_{i=1}^{n}\prod_{k=1}^{K} k_i *p_k(\textbf x_i) = \frac{-1}{n}\sum_{i=1}^{n}\sum_{k=1}^{K} {k_i} * log(p_k(\textbf x_i))$$

Finally, we introduce the L2 norm to obtain a regularized objective function.

\begin{equation} J(\textbf w) = \frac{-1}{n}\sum_{i=1}^{n}\sum_{k=1}^{K} {k_i} * log(p_k(\textbf x_i)) + \alpha R(\textbf w) = \frac{-1}{n}\sum_{i=1}^{n}\sum_{k=1}^{K} {k_i} * log\bigg(\frac{e^{\beta_{0}^k + \textbf w_k\cdot \textbf x_i}}{\sum_{k=1}^{K} e^{\beta_{0}^k + \textbf w_k\cdot \textbf x_i}}\bigg)+ \alpha R(\textbf w) \end{equation}

where $R(\textbf w) = ||w||^2_2 = \sum^{k}_{i=1}w_i^2$

Derivative of Softmax¶

Given a n-dimensional column vector $\textbf z$, we can calculate the derivative of the $j$th component of the output vector with respect to the kth component of the input vector $\textbf z$. Once we compute this, we can form the Jacobian matrix for the general derivative \begin{equation} S(\textbf z_k) = \frac{e^{\textbf z_k}}{\sum_{i=1}^{K} e^{\textbf z_i}} \end{equation}

If k==j: \begin{equation} \frac{\partial S_j}{\partial z_k} = \frac{\partial}{\partial z_k}\bigg[\frac{e^{\textbf z_k}}{\sum_{i=1}^{K} e^{\textbf z_i}}\bigg] = \frac{e^{\textbf z_k}\big(\sum_{i=1}^{K} e^{\textbf z_i}\big) - e^{\textbf z_k}e^{\textbf z_j}}{\big(\sum_{i=1}^{K} e^{\textbf z_i}\big)^2} = \frac{e^{\textbf z_k}}{\big(\sum_{i=1}^{K} e^{\textbf z_i}\big)}\bigg(1-\frac{e^{\textbf z_j}}{\big(\sum_{i=1}^{K} e^{\textbf z_i}\big)}\bigg) = S_k(1-S_j) \end{equation}

If k!=j

\begin{equation} \frac{\partial S_j}{\partial z_k} = \frac{\partial}{\partial z_k}\bigg[\frac{e^{\textbf z_k}}{\sum_{i=1}^{K} e^{\textbf z_i}}\bigg] = \frac{ - e^{\textbf z_k}e^{\textbf z_j}}{\big(\sum_{i=1}^{K} e^{\textbf z_i}\big)^2} = -S_kS_j \end{equation}

And more concisely, \begin{equation} \frac{\partial S_j}{\partial z_k} = S_k(\delta_{kj}-S_j) \end{equation}

Gradient of Cost Function¶

$J(\textbf w)$ can be simplified(thought as) with a vectorized version of the softmax function taking in the data matrix $X$ (nxd) multipled by the weight matrix $\textbf w$(dxk). This would produce a nxk matrix where each row is a list of K probabilities associated with the ith observation.

Let the matrix transformations described above be $L: \mathbb{R}^{nd} x \mathbb{R}^{dk} \rightarrow \mathbb{R}^{nk}$ be the first step and the softmax layer be $S: \mathbb{R}^{nk} \rightarrow \mathbb{R}^{nk}$

Then,

\begin{equation} \frac{dJ}{dw} = \frac{dJ}{dS} \frac{dS}{dw} \end{equation}

The gradient of $J(\textbf w)$ is a vector of partials with respect to $\textbf w_1,.,\textbf w_j,..,\textbf w_K$ \begin{equation} J(\textbf w) = \frac{-1}{n}\sum_{i=1}^{n}\sum_{k=1}^{K} k_i * \bigg(log\bigg(e^{\beta_{0}^k + \textbf w_k\cdot \textbf x_i}\bigg) - log \bigg(\sum_{k=1}^{K} e^{\beta_{0}^k + \textbf w_k\cdot \textbf x_i}\bigg)\bigg)+ \alpha R(\textbf w) \end{equation}

\begin{equation} \frac{\partial J(\textbf w)} {\partial \textbf w_j} = \bigg(\frac{-1}{n}\sum_{i=1}^{n} j_i * \textbf x_i\bigg) + \bigg(\sum_{i=1}^{n} j_i * \frac{e^{\textbf w_j \cdot \textbf x_i}* \textbf x_i}{\sum_{k=1}^{K} e^{\beta_{0}^k + \textbf w_k\cdot \textbf x_i}}\bigg)+ \alpha * \textbf w_k \end{equation}\begin{equation} \frac{\partial J(\textbf w)} {\partial \textbf w_j} = \bigg(\frac{1}{n}\sum_{i=1}^{n} \textbf x_i * (j_i - p_j(\textbf x_i)\bigg) + \alpha * \textbf w \end{equation}

Gradient Descent Optimizations - Momentum, Nesterov Accelerated Gradient, RMSProp¶

To avoid the problem of in "...minimizing highly non-convex error functions ... for neural networks is avoiding getting trapped in their numerous suboptimal local minima",these three optimizations rely on the idea of exponentially weighted averages. That is

$$\upsilon_t = \gamma * \upsilon_{t-1} + (1-\gamma) * \nabla_\theta J(\theta)$$$$ \theta_{t} = \theta_{t-1} - \alpha * \upsilon_t$$

where $\gamma$ is typically .9 and intuitively translates to a moving average over the past 10 iterations. When the gradient changes directions, a smaller gamma value will increase the change in updating the parameters and vice-versa. Momentum allows to keep going, not in the direction of steepest descent, but to move quickly with smaller and consistent gradients.

NAG¶

If we want our momentum to be smarter, to slow down as it approaches a changing topology, we evaluate the gradient at the future value of our parameters, a look-ahead. This prevents the gradient from changing too fast when it really should not.

$$\upsilon_t = \gamma * \upsilon_{t-1} + (1-\gamma) * \nabla_\theta J(\theta - \gamma * \upsilon_{t-1})$$$$ \theta_{t} = \theta_{t-1} - \alpha * \upsilon_t$$
RMSProp¶

This optimization introduces the idea of dividing the gradient by a running average of its magnitude. This idea is highly related to adapting the learning rate to each of the parameters $w_i$, like Adagrad, Adadelata do.

$$\upsilon_t = \gamma * \upsilon_{t-1} + (1-\gamma) * \nabla_\theta J(\theta)^2$$$$ \theta_{t} = \theta_{t-1} - \alpha * \frac{\nabla_\theta J(\theta)}{\sqrt{\upsilon_t + \epsilon}}$$

where $\epsilon$ is a very small number for numerical stability.

class GradientDescent:
    def __init__(self):
        self.rate = .1
        self.precision = .0001
        self.reg = .1
    
    # Pre: X - design matrix nxd
    #      y - labeled classes 
    #      w - pxk parameter matrix of current model
    # Post: a vector of K errors,one for each label
    def determineCost(self, X, y, w):
        # yhat is a nxK matrix, where each row is the probability across all output classes
        yhat = self.softmax(np.dot(X,w))
        # sum K sets of squared w_k
        R_w = self.reg * np.sum(w * w, axis = 0)
        # * operator will ensure a valid summation over all samples because the labels have been one-hot encoded
        J = -np.sum((y * np.log(yhat)),axis = 0)+R_w
        return J/X.shape[0]
    
    # Pre: X - design matrix
    #      y - labeled classes
    #      w - pxk parameter matrix of current model
    # Post: the gradient of J evaluated at w
    def gradient(self, X, y, w,batch_size):
        ind = np.random.choice(X.shape[0], batch_size-1, replace=False)
        X_sample = X[ind, :]
        y_sample = y[ind,:]
        yhat = self.softmax(np.dot(X_sample,w))
        gradient = np.dot(X_sample.transpose(),(yhat-y_sample))+self.reg * w
        return gradient/X_sample.shape[0]
    
    # Pre: z = x mult w of dimension (n*p)multiply(p*K=10)=>n*10
    # Post: Softmax score for each observation
    def softmax(self,z):
        return np.exp(z-np.max(z)) / np.sum(np.exp(z-np.max(z)), axis=1, keepdims=True)
    
    def error(self,X, y, w):
        yhat = self.softmax(np.dot(X,w))
        return np.sum(np.argmax(yhat,axis=1)!=np.argmax(y,axis=1))/float(X.shape[0]) 
    
    # Pre: X - design matrix
    #      y - labeled classes
    #      w - pxk parameter matrix of current model
    # Post: return optimal set of weights w, J(theta) over training set,accuracy over training set
    def gradientDescent(self,X,y,w,batch_size):
        rate = self.rate
        iteration = 0
        J = []
        accuracy = []
        J.append(np.sum(np.array([0]*10)))
        J.append(np.sum(self.determineCost(X,y,w)))
        while(np.abs(J[-1]-J[-2]) > self.precision):
            if((len(J)-1)%10==0):
                accuracy.append(1-self.error(X,y,w))
            w = w - rate * self.gradient(X,y,w,batch_size)
            J.append(np.sum(self.determineCost(X,y,w)))
            iteration = iteration + 1
            rate = rate * 1/(1 + .0001 * iteration)
        return w,J,accuracy

from sklearn.preprocessing import OneHotEncoder
class LogisticRegression:
    def __init__(self):
        self.GD = GradientDescent()
        self.w = []
        self.accuracy = []
        
    # Pre - data matrix X
    # Post - data matrix X with bias term concatenated as first column 
    def addBias(self,X):
        X = np.concatenate((np.ones((X.shape[0],1)),X),axis=1)
        return X
    
    def normalize(self,X):                                                     
        X = (X - np.min(X))/float(np.max(X) - np.min(X))
        return X
    
    def oneHotEncode(self,y):
        enc = OneHotEncoder(handle_unknown='ignore')
        enc.fit(y)
        return np.array(enc.transform(y).toarray())
    
    # trains two models, with and without batch gradient descent
    def train(self,trainX,trainy,experiment = False):
        trainX = self.normalize(trainX)
        trainX = self.addBias(trainX)
        trainy = self.oneHotEncode(trainy)
        
        p,k = trainX.shape[1],trainy.shape[1]
        w = np.random.rand(p,k)
        
#         Without Batch
        if experiment == False:
            print("Training Started")
            start_time = time()
            w,J,accuracy = self.GD.gradientDescent(trainX,trainy,w,trainX.shape[0])
            self.w.append(w)
            self.accuracy.append(accuracy)
            end_time = time()
            print("Training Finished, Took: %ss"%((end_time-start_time)))
        else:
            # With Batch
            print("Training Started with Batch Gradient Descent for batch sizes of size ",32,"to",trainX.shape[0]/2)
            w = w_orig = np.random.rand(p,k)
            for i in range(32, int(trainX.shape[0]/2), 512):
                start_time = time()
                w,J,accuracy = self.GD.gradientDescent(trainX,trainy,w,i)
                end_time = time()
                print("Training Finished with Batch Gradient Descent for batch size = ,",i,", Took: %ss"%((end_time-start_time)))
                self.w.append(w)
                self.accuracy.append(accuracy)
                w = w_orig
        
        return self.w,self.accuracy
    
    def test(self,testX,testy,w):
        testX = self.normalize(testX)
        testX = self.addBias(testX)
        testy = self.oneHotEncode(testy)
        return 1 - self.GD.error(testX,testy,w)

        J.append(np.sum(self.determineCost(X,y,w)))
        upsilon = 0
        while(np.abs(J[-1]-J[-2]) > self.precision):
            if((len(J)-1)%10==0):
                accuracy.append(1-self.error(X,y,w))
            upsilon = (self.gamma * upsilon) + rate * self.gradient(X,y,w,batch_size,upsilon)
            w = w - upsilon
            J.append(np.sum(self.determineCost(X,y,w)))
            iteration = iteration + 1
            rate = rate * 1/(1 + .0001 * iteration)
        return w,J,accuracy
